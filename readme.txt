This zip file contains four subfiles each including the code and data (csv file) for our four different simulations.
L1 -- Logistic data with 6 variables
L2 -- Logistic data with 32 variables
T1 -- Arctan data with 6 variables
T2 -- Arctan data with 32 variables

This zip file also contains a rmd file call data_generate.rmd to generate all the data we used in this project.
